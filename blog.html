What is DOM?

The DOM (Document Object Model) is a programming interface for web documents. It represents the structure of a webpage so that programs can interact with it, modify its content, structure, and styles

what is event delegation in the context of DOM, and why is it useful?

Event delegation is a technique in the DOM where you take advantage of event bubbling (or event propagation) to handle events at a higher-level element (parent element), rather than attaching individual event listeners to child elements.Event delegation reduces memory usage and improves performance, handle dynamic elements.
